Hippo Run

About:
The idea of this project is to utilize Javascript's Web Speech API to create a simple dodging game that takes voice inputs.
Instructions:
Hey, there's a hippo in danger! Go help him out by telling him where to go or moving him yourself.

Please look at controls on the right for available commands.

Voice: You will have to allow mic access on your browser! FYI, sometimes the hippo just doesn't want to listen. Sorry!

Keyboard: Please use ASD to move.
